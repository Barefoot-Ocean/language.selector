# language.selector 0.1.0

Initial CRAN release.
## Features

* `select_languageInput()` - Interactive language selector component
  - Responsive design with desktop horizontal menu and mobile modal popup
  - Built-in flag icons for 10 languages (en, ceb, es, fil, fr, id, mg, pt, sw, tet)
  - Click outside to close on both desktop and mobile
  - Smooth CSS animations

* `updateSelect_languageInput()` - Update selector from server

* `language_flags` - Data frame of available built-in language flags

* `languages` - Pre-configured list of language options

* `generate_icon_css()` - Helper function to generate CSS for custom flag icons

## Built-in Languages

- English (en)
- Cebuano (ceb)
- Spanish (es)
- Filipino (fil)
- French (fr)
- Indonesian (id)
- Malagasy (mg)
- Portuguese (pt)
- Swahili (sw)
- Tetum (tet)
